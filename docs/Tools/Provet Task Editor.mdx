# Provet Task Editor

This page embeds a small in-browser editor for Provet reminder tasks (e.g. `appointmentReminders`, `sendReminders`) so you can tweak schedules/params and export valid JSON.

:::caution[Privacy / credentials]
Anything you paste here stays in your browser (it’s a static page with no backend calls), but exported JSON may contain sensitive values like WhatsApp `accountId`. Don’t paste secrets into screenshots or public issues.
:::
