Premium Homepage + Theme Polish

April 29, 2026

A refreshed premium landing experience for Texter Docs, with a redesigned homepage, improved dark/light theme cohesion, and a better navbar experience.

Added​

  • Homepage hero redesign — premium landing hero with glow + subtle animated grid background and polished CTA buttons.
  • Homepage feature cards — new glass/white card grid with icons, hover lift, and clearer hierarchy linking into key docs areas.
  • Navbar search — search is now available directly in the top navbar.

Improved​

  • Dark + light theme cohesion — updated shared theme tokens so switching themes keeps the homepage and cards looking consistent and premium.
  • Scenario Marketplace visuals — improved card hover shadow to follow the active theme primary color.
  • Scenarios header (dark mode) — upgraded the hero background to match the new dark aesthetic.

Fixed​

  • Search bar overlap — removed the extra keyboard badge styling that conflicted with the search UI.
